# U.S. States Game

Welcome to the U.S. States Game! Test your knowledge of U.S. geography by guessing the names of the states.

## Overview

This simple Python-based game uses the Turtle graphics library and pandas to create an interactive experience where users can input state names, and the program will plot them on a map. The game is designed to help users learn and remember the names and locations of all 50 U.S. states.

## How to Play

1. Run the script in a Python environment.
2. A window will open with a blank map of the U.S.
3. Enter the names of the states when prompted.
4. The program will plot the guessed states on the map.
5. Type "Exit" to end the game and generate a list of states you need to learn.

## Prerequisites

- Python
- Turtle graphics library
- pandas library
